## Overview

**NTE-scripts** is a small collection of scripts and tooling created for experimenting with the Android version of **NTE**.

The goal of this repository is to provide a clean, organized, and transparent place for storing scripts, notes, utilities, and related resources.  
This project is currently in an early stage, so the structure and contents may change over time.

> Built for research, learning, testing, and personal customization.
