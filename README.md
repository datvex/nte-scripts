## Обзор

**NTE-scripts** — это небольшая коллекция скриптов и вспомогательных инструментов, созданных для экспериментов с Android-версией **NTE**.

Цель этого репозитория — предоставить аккуратное, понятное и открытое место для хранения скриптов, заметок, утилит и связанных материалов.  
Проект находится на ранней стадии разработки, поэтому структура, содержимое и документация могут изменяться со временем.

> Создано для исследования, обучения, тестирования и личной настройки.

Текущее предполагаемое содержимое:

| Раздел | Описание |
